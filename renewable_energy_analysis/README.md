# 🌍 Renewable Energy Worldwide Analysis (1965–2022)

## 📌 Project Overview

This project analyzes global renewable energy adoption trends from **1965 to 2022** using historical energy data. The analysis explores how renewable energy has evolved across countries, identifies growth patterns, examines policy impacts, and forecasts future renewable energy adoption using **time-series analysis and linear regression**.

The project focuses on understanding:

* Global renewable energy growth trends
* Country-level adoption patterns
* Fastest-growing renewable energy nations
* Policy impact analysis (Kyoto Protocol & Paris Agreement)
* Forecasting the transition to **100% renewable energy**

---

## 📊 Dataset Information

**Dataset:** Renewable Energy Worldwide (1965–2022)

The dataset includes global renewable energy production share across multiple countries and regions.

### Dataset Columns

| Column                                   | Description                          |
| ---------------------------------------- | ------------------------------------ |
| Entity                                   | Country or region name               |
| Code                                     | Three-letter country code            |
| Year                                     | Year of observation                  |
| Renewables (% equivalent primary energy) | Percentage of renewable energy share |
